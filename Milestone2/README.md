# Employee Wellness Management Analytics

## Milestone 2 – NLP Preprocessing Pipeline

### Project Objective

The objective of this milestone is to develop an efficient Natural Language Processing (NLP) preprocessing pipeline for the Employee Wellness Management Analytics (MoodMentor) application. The pipeline processes multilingual employee feedback by performing language detection, text cleaning, normalization, tokenization, sentence segmentation, translation, lemmatization, emoji handling, and preprocessing statistics generation. The processed text is then ready for sentiment analysis and emotion detection.

## Google Colab Setup Instructions

1. Open **NLP_Preprocessing_Pipeline.ipynb** in Google Colaboratory.
2. Install all required Python libraries.
3. Download the required NLTK resources and spaCy language model.
4. Upload the employee feedback dataset (CSV/TXT).
5. Run all notebook cells sequentially.
6. View the generated preprocessing outputs and statistics.
